## Getting Started

### Prerequisites

- Have deno installed (for running in local)
- Click this link to install deno at your machine
  https://deno.land/manual@v1.29.4/getting_started/installation
- Clone the repo

  ```
  git clone https://github.com/RealtechLtd/poc-deno.git
  ```
